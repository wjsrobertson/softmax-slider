Softmax Slider
==============

A visual aid for playing with the [SoftMax](https://en.wikipedia.org/wiki/Softmax_function) algorithm.

### Live Demo

You can play with it in action here: [xylophones.net/softmax-slider](http://xylophones.net/softmax-slider/)
